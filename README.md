# Projeto Av3 Banco de Dados
# Sistema de Biblioteca (Cenário 04)

Este repositório contém a modelagem e os scripts da base de dados de um Sistema de Biblioteca.

## 📁 Estrutura

* **`./script/`**: Diretório que centralizará os scripts para carga e consulta de dados SQL.
* **`./script/carga`**: Diretório que centralizará os script de carga do Banco.
* **`./script/consulta`**: Diretório que centraliszará as consultas do projeto.
* **`./DER_Visual.excalidraw`**: Diagrama Entidade-Relacionamento.
* **`./MER_Visual.json`**: Modelo Entidade-Relacionamento.

## Como Visualizar os Dados
### Diagrama Entidade-Relacionamento (DER)
Para visualizar os dados do DER, será necessário ter a extensão `ERD Editor` do github, feito isso, basta abrir o arquivo normalmente que os dados serão exibidos.
